### Checkout

:computer: Checkout this repo, install dependencies, then start the http-server process with the following:

```
	> git clone git@github.com:AceDZN/ReduxGulpStarterKit.git
	> cd ReduxGulpStarterKit
	> npm install
	> gulp
```
